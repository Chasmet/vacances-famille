# Vacances Famille

Application personnelle de gestion des vacances pour deux profils : Moi et Ma femme.

Fonctions principales :

- Calcul automatique des jours ouvrés du lundi au vendredi.
- Deux semaines comptent 10 jours.
- Calendrier séparé pour chaque profil.
- Onglets : Vac 1, Vac 2, Vac 3, RTT, Pont, Sans solde, Autre.
- Jours fériés français exclus si l’option est activée.
- Quotas annuels réglables.
- Détection des chevauchements.
- Sauvegarde locale sur téléphone.
- Export et import JSON.
- Installation Android via GitHub Pages.
